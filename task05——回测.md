# 什么是回测？
用历史的数据跑策略，在真实场景中不一定有效

# 如何模拟交易
由策略生成仓位
position=1:持仓
position=0:空仓

# 策略的收益率
由股票的价格得到**日收益率**，再由**日收益率**计算**策略日收益**
**策略日收益**=**仓位**✖️**日收益率**
每日收益连乘=累计收益曲线（净值曲线）
基准曲线：同标的一直持仓，同期持有大盘

# 回测结果可视化
关注：
1. 整体趋势
2. 对比基准曲线
3. 回撤

# 胜率
整个交易回合中：赢➗（赢➕输）
赢：价格上涨的次数
输：价格下跌的次数

# 最大回撤
亏的程度最大
